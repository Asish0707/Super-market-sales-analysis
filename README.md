📊 Features
Generates a simulated supermarket sales dataset
Performs data analysis using:
Pandas for data handling
NumPy for numerical operations
Creates visualizations using:
Matplotlib
Seaborn
📁 Dataset Details

The dataset includes fields such as:

Invoice ID
Customer Type
Gender
Product Line
Total Purchase Amount
Hourly sales data

Note: The dataset is randomly generated using numpy.random.

📈 Visualizations Included
1. Revenue by Product Line
Bar chart showing total revenue for each product category
2. Average Spending by Gender
Compares average purchase amounts between genders
3. Customer Type Distribution
Pie chart showing proportion of customer types
4. Hourly Sales Trend
Line chart showing how sales vary by hour
5. Correlation Heatmap
Displays relationships between numerical features
6. Total Bill Distribution
Histogram with KDE showing distribution of total purchase amounts
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
▶️ How to Run
Install required libraries:
pip install pandas numpy matplotlib seaborn
Open the notebook:
jupyter notebook "Super market sales .ipynb"
Run all cells to generate dataset and visualizations.
